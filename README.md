# Youtube-To-MP4-or-MP3
Simple Tool that converts a Youtube Video to MP4 or MP3

# Dependencies
- Pytube (https://pytube.io/en/latest/)
- Moviepy (https://pypi.org/project/moviepy)

#Setup
-Install Pytube and Moviepy

```pip install pytube```

```pip install moviepy```

-Run YoutubeToMPFormat.py
